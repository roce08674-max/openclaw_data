## 快速开始

### 基本概念

文章结构拆解agent负责文章结构设计、叙事节奏指导、模板库管理。一个好的结构是文章逻辑清晰、阅读体验流畅的基础。

### 工作流程

```
内容分析 → 结构选择 → 节奏规划 → 提纲撰写 → 质量检查
```

### 1. 观点输出型

**适用场景**: 观点类、评论类、思考类文章
**篇幅建议**: 1500至2500字
**核心模式**: 总分总模式

│ 下降到              │

└─────────────────────┘

【正文】（1200至1800字）
┌─────────────────────┐
│ 分论点一加案例支撑   │
│ （400至600字）       │
├─────────────────────┤
│ 过渡段               │
│ （100至150字）       │
├─────────────────────┤
│ 分论点二加案例支撑   │
│ （400至600字）       │
├─────────────────────┤

├─────────────────────┤

## 叙事节奏指南

### 什么是叙事节奏

叙事节奏是指文章内容展开的快慢、起伏、高低的规律性变化。好的节奏能够：
- 影响阅读体验
- 决定读者是否能读下去
- 决定文章的传播效果

### 核心原则

- 有张有弛：紧张和放松交替
- 有起有伏：高潮和铺垫配合
- 有快有慢：重要内容放慢节奏
- 有轻有重：关键点重点强调

---

### 节奏控制技巧

#### 开头节奏选择

| 节奏类型 | 特点 | 适用场景 |
|---------|------|---------|
| 快速切入 | 直奔主题，不拖沓 | 干货类、观点类 |
| 悬念引入 | 设置疑问，引发好奇 | 故事类、热点类 |
| 场景描写 | 营造氛围，代入感强 | 情感类、故事类 |
| 数据冲击 | 用数据震撼读者 | 深度分析、调查报告 |

#### 正文节奏控制

**铺垫段（慢）**: 详细描述背景，建立读者认知，为高潮做准备

**过渡段（中）**: 连接前后内容，承上启下，转换话题

**重点段（慢）**: 核心观点、重要案例、关键数据，需要放慢节奏详细展开

**说明段（中）**: 解释说明、补充信息、举例论证

**过渡到高潮（快）**: 加快叙述速度，制造紧张感，积蓄力量

#### 高潮节奏设计

**高潮前（慢）**: 渲染气氛，埋下伏笔，制造期待

**高潮中（极慢）**: 详细描写，多角度呈现，情感释放

**高潮后（快）**: 迅速收尾，转向结局，避免拖沓

---

### 节奏加速技巧

**省略法**: 删除冗余的细节描述，让叙述更紧凑

```
错误示例：他回到家里，脱下鞋子，坐在沙发上，打开电视
正确示例：他回到家，打开电视
```

**概括法**: 用简洁的语言概括一段时间内的行为

```
错误示例：那段时间他每天早上六点起床
正确示例：那段时间，他每天早起
```

**跳写法**: 跳过不重要的过程，直接跳到结果

```
错误示例：他去了超市，买了菜，回家做饭
正确示例：一天很快过去了
```

---

### 节奏减速技巧

**细节描写**: 用丰富的细节增强画面感

```
普通：他很紧张
细节：他的额头渗出汗珠，手心攥紧了衣角
```

**对话展示**: 用对话展现人物内心和情节发展

```
普通：他对我说了一些话
对话："你真的想好了吗？"他问我
```

**心理描写**: 展现人物的内心活动

```
普通：她很纠结
心理：去还是不去？她反复问自己这个问题
```

**环境渲染**: 用环境描写烘托氛围

```
普通：天气很冷
环境：北风呼啸，雪花打在脸上，像刀割一样疼
```

### 不同文体的节奏特点

#### 观点输出型

```
开头（快）：引入话题，直奔观点
中间（慢）：逐条展开，详细论证
结尾（慢）：总结升华，行动号召

节奏曲线：快 → 慢 → 快 → 慢
```

#### 故事叙述型

```
开头（慢）：设置悬念，引人入胜
发展（快）：事件推进，情节展开
高潮（极慢）：重点描写，情感释放
结尾（中）：快速收尾，升华主题

节奏曲线：慢 → 快 → 极慢 → 快
```

#### 干货分享型

```
开头（快）：痛点引入，解决方案预告
中间（中）：方法步骤，清晰展开
结尾（慢）：重点总结，行动号召

节奏曲线：快 → 中 → 慢
```

#### 热点解读型

```
开头（快）：事件简述，快速切入
中间（慢）：深度分析，多角度解读
结尾（中）：总结观点，引发思考

节奏曲线：快 → 慢 → 中
```

### 节奏调整技巧

#### 段落长短交替

```
长段落（三至五行）：详细说明、重点内容
短段落（一至两行）：强调重点、转折过渡

示例：
长：他从小就热爱学习，每天放学后都会去图书馆。周末的时候，他更是会花一整天时间阅读各种书籍（背景铺垫）

短：但命运的转折点，出现在他十四岁那年（转折）
```

#### 句式变化

**长句**: 详细表达、完整阐述

**短句**: 强调重点、节奏感强

```
长句：那天，他在图书馆遇到了一个改变他一生的人，那个人递给他一本书，书名是《时间简史》

短句：就这样，他遇见了那个人，那本书，那个改变一切的瞬间
```

#### 修辞手法

- **排比句**: 增强气势、节奏感
- **对比句**: 突出差异、强调重点
- **比喻句**: 形象生动、便于理解
- **设问句**: 引发思考、增强互动

## 结构选择决策

### 决策流程图

```
文章内容 → 核心目的 → 选择结构

如果：核心目的是表达观点 → 选择"观点输出型"
如果：核心目的是传授方法 → 选择"干货分享型"
如果：核心目的是讲述故事 → 选择"故事叙述型"
如果：核心目的是解读事件 → 选择"热点解读型"
如果：核心目的是整理资源 → 选择"清单盘点型"
如果：核心目的是对比分析 → 选择"对比分析型"
```

### 选择因素考量

**内容类型**: 纯观点、干货教程、故事、热点、资源整理还是对比分析

**目标读者**: 读者群体是什么类型，他们习惯什么样的阅读方式

**传播目的**: 是要表达观点、传递知识、还是引发讨论

**篇幅要求**: 根据目标平台的字数限制选择合适的结构
