## 快速开始

### 基本概念

公众号文章agent负责公众号平台专属写作规范、风格指导、模板管理。公众号文章有独特的写作要求，需要符合平台规则和读者习惯。

### 工作流程

```
内容分析 → 风格适配 → 模板选择 → 写作执行 → 合规检查
```

## 写作风格指南

### 公众号文章核心特征

#### 1. 口语化表达

公众号文章不同于传统媒体，需要更加口语化、接地气的表达方式。

```
错误示例：关于该产品的市场前景，我们需要进行深入分析
正确示例：这款产品到底值不值得买？今天我们来聊聊

错误示例：该研究报告表明，消费者行为呈现出明显的趋势性变化
正确示例：最近的一份报告说，消费者买东西的方式变了
```

**转换技巧**：

将书面语转换为口语时，可以遵循以下原则：
- 用"说白了"代替"综上所述"
- 用"这么说吧"代替"由此可见"
- 用"所以"代替"因此"
- 用"总的来说"代替"综上所述"

#### 2. 网感与年轻化

适度使用网络热词，善用表情和emoji，符合年轻用户的阅读习惯。

**常用网感表达**：

| 表达类型 | 示例 |
|---------|------|
| 开头引入 | 家人们，谁懂啊... |
| 情感共鸣 | 这也太香了吧、打工人破防了 |
| 强调表达 | 绝绝子、YYDS、破防了 |
| 互动表达 | 评论区聊聊、你怎么看 |
| 结尾互动 | 看到这个你想到了什么 |

**使用建议**：网感表达要适度，过度使用会显得刻意和不专业。根据目标读者群体调整使用频率。

#### 3. 情感共鸣

公众号文章需要与读者产生情感连接，从用户痛点切入，讲述真实故事，引发读者情感认同。

**情感共鸣切入点**：

| 切入类型 | 示例 |
|---------|------|
| 职场痛点 | 凌晨三点，你还在加班吗？ |
| 生活共鸣 | 那个在北京打拼的你，累了吧 |
| 情感表达 | 有没有一瞬间，你突然想放弃了 |
| 成长经历 | 那个裸辞去大理的姑娘，后来怎么样了 |

**情感表达技巧**：

- 从读者关心的问题入手
- 使用第二人称"你"拉近距离
- 描述具体场景，引发画面感
- 表达真实感受，不矫揉造作

#### 4. 实用价值

公众号文章需要给读者提供实际价值，提供干货知识，解决具体问题，分享独到观点。

**实用价值体现方式**：

| 价值类型 | 示例 |
|---------|------|
| 技巧方法 | 三个技巧，让你的效率提升十倍 |
| 工具推荐 | 新手做自媒体，必备工具清单 |
| 经验分享 | 月入过万的副业推荐 |
| 知识科普 | 一篇文章讲清楚XXX |

### 语言风格特点

#### 开头写作技巧

好的开头能够在三秒内抓住读者注意力，引导读者继续阅读。

**开头类型与示例**：

**场景引入型**：
```
上周，我一个朋友跟我说，他想离职。
```

**问题引导型**：
```
你有没有发现，身边越来越多的人，开始不愿意发朋友圈了？
```

**数据冲击型**：
```
二零二四年第一个季度刚过，我就发现了一个可怕的规律。
```

**故事切入型**：
```
三年前，他还是一个月薪五千的普通员工...
```

#### 过渡句技巧

好的过渡能够让文章逻辑更流畅，阅读体验更连贯。

**常用过渡句式**：

| 过渡类型 | 示例 |
|---------|------|
| 转折过渡 | 但问题是...、其实不只是这样... |
| 递进过渡 | 说到这儿...、你可能会问，那应该怎么办？ |
| 对比过渡 | 刚才说的是A，但B也同样重要。 |
| 因果过渡 | 之所以会这样，是因为... |

#### 金句打造技巧

金句是文章的亮点，能够增强传播力，让读者记住核心观点。

**金句标准**：

- 简短有力（十至二十字）
- 引发共鸣
- 便于传播
- 有独特观点

**金句示例**：

| 金句类型 | 示例 |
|---------|------|
| 观点类 | 不是努力没用，而是努力的方式不对。 |
| 感悟类 | 成年人的崩溃，往往就在一瞬间。 |
| 选择类 | 选择比努力更重要，但努力是为了有更多选择。 |
| 成长类 | 你现在的选择，决定了三年后的自己。 |

**金句打造方法**：

- 提炼核心观点
- 用简短的语言表达
- 加入情感元素
- 创造记忆点

#### 口语化表达技巧

**技巧一：少用书面语**

```
错误示例：因此综上所述由此可见
正确示例：所以总的来说这么说吧

错误示例：由于受到多种因素的影响，该产品的销售情况呈现出一定程度的波动
正确示例：说白了，这个产品卖得一般
```

**技巧二：多用短句**

长句拆分成短句，阅读更流畅。

```
错误示例：由于受到多种因素的影响，该产品的销售情况呈现出一定程度的波动
正确示例：说白了，这个产品卖得一般，确实不太理想。
```

**技巧三：加入语气词**

适当加入语气词，增加亲和力。

```
正确示例：说实话...、坦白说...、你发现了没...
```

**技巧四：适当设问**

引发读者思考，增强互动感。

```
正确示例：你有没有想过...、为什么...呢？、如果...会怎样？
```

## 文章模板库

### 模板一：观点输出型

**适用场景**：观点类、评论类、思考类文章

**模板结构**：

```
【标题】
吸引眼球的观点式标题

【开头】（一百至一百五十字）
- 用一个场景或故事引入
- 提出核心观点
- 预告文章价值

【正文】（一千五百至二千五百字）
- 观点一：分论点加案例支撑
- 过渡段落
- 观点二：分论点加案例支撑
- 过渡段落
- 观点三：分论点加案例支撑

【结尾】（一百至二百字）
- 总结核心观点
- 升华主题
- 引导互动（评论、转发）
```

**写作要点**：

- 观点要鲜明独特，有自己的见解
- 每个观点都要有案例支撑
- 案例要具体，最好有数据或细节
- 结尾要有升华，给读者启发

---

### 模板二：干货分享型

**适用场景**：教程类、技巧类、工具类文章

**模板结构**：

```
【标题】
数字加痛点加解决方案的标题

【开头】（一百字）
- 描述痛点场景
- 介绍文章将提供什么解决方案
- 预告文章价值

【正文】（二千至三千字）
- 问题背景
- 解决方案一（详细步骤加注意事项）
- 解决方案二（详细步骤加注意事项）
- 解决方案三（详细步骤加注意事项）
- 常见误区或坑点提醒

【结尾】（一百字）
- 总结核心要点
- 鼓励行动
- 资源推荐
```

**写作要点**：

- 解决方案要具体可操作
- 每个方法都要有步骤说明
- 注意事项要实用
- 常见误区要有针对性

---

### 模板三：故事叙述型

**适用场景**：人物故事、案例分享、经历类文章

**模板结构**：

```
【标题】
带有情感张力的故事标题

【开头】（一百五十字）
- 制造悬念或冲突
- 预告故事走向
- 吸引读者继续阅读

【正文】（二千至三千字）
- 背景铺垫
- 事件发展（起承转合）
- 高潮转折
- 结局与感悟

【结尾】（一百五十字）
- 故事寓意
- 联系读者
- 行动号召
```

**写作要点**：

- 开头要有悬念，抓住读者好奇心
- 故事发展要有起承转合
- 高潮部分要有情感张力
- 结尾要有升华，联系读者生活

---

### 模板四：热点解读型

**适用场景**：热点分析、事件解读、趋势分析文章

**模板结构**：

```
【标题】
事件或趋势加独特视角的标题

【开头】（一百五十至二百字）
- 热点事件简述
- 说明分析角度
- 引出核心观点

【正文】（一千五百至二千五百字）
- 事件回顾（时间线）
- 原因分析（多角度）
- 影响评估（短期与长期）
- 趋势预测
- 应对建议

【结尾】（一百五十字）
- 总结要点
- 预测未来
- 引导思考
```

**写作要点**：

- 事件回顾要清晰，有时间线
- 分析要有深度，多角度
- 预测要有依据，不是凭空想象
- 建议要实用，有可操作性

---

### 模板五：清单盘点型

**适用场景**：资源整理、工具推荐、榜单类文章

**模板结构**：

```
【标题】
最全或必备加主题的标题

【开头】（一百至一百五十字）
- 说明整理原因
- 介绍资源类型
- 预告内容丰富

【正文】（二千至三千字）
- 分类一：资源一加使用说明加获取方式
- 分类二：资源二加使用说明加获取方式
- ...
- 分类N：资源N加使用说明加获取方式

使用建议：
- 入门建议
- 进阶建议
- 避坑指南

【结尾】（一百至一百五十字）
- 总结资源价值
- 告知更新计划
- 互动引导
```

**写作要点**：

- 资源要实用，有使用价值
- 每个资源要有使用说明
- 获取方式要具体
- 使用建议要有针对性

---

### 模板六：深度解读型

**适用场景**：深度分析、专业话题解读

**模板结构**：

```
【标题】
关于加主题，这是我看过的最深度分析

【开头】（一百五十至二百字）
- 描述现象
- 说明分析角度
- 预告分析内容

【正文】（二千至三千字）
- 现象回顾（时间线）
- 原因分析（多角度）
- 影响评估（短期与长期）
- 趋势预测
- 应对建议

【结尾】（一百五十字）
- 总结要点
- 预测未来
- 引导思考
```

## 敏感词库

### 违禁词类别

#### 绝对化用语

```
绝对、最好、第一、顶级、唯一、顶尖、史上最强、绝对安全、绝对有效、百分之百
```

#### 夸张用语

```
震惊、全惊呆、吓尿、哭晕、必看、必读、必须知道、惊呆所有人、震撼全网、轰动全球
```

#### 虚假承诺

```
一百%有效、无效退款、一招见效、保证赚钱、包治百病、七天学会、终身受益
```

#### 违规承诺

当用于虚假宣传时，以下词汇也属于违规：
```
独家、揭秘、内幕、真相、惊天大秘密、内部消息
```

### 平台禁忌

#### 微信公众平台

- 禁止标题党
- 违规会限流
- 虚假宣传会封号
- 侵权内容会被投诉

#### 内容红线

- 政治敏感：不评价领导人、不涉及政治立场
- 违法违规：不传播不实信息、不教唆犯罪
- 侵权盗版：不抄袭他人内容、不使用未授权图片
- 虚假宣传：不夸大产品功效、不做虚假承诺

### 敏感词检测方法

**方法一：逐词检查**

对照敏感词库，逐词检查文章内容，标记并替换敏感词。

**方法二：工具检测**

使用敏感词检测工具进行批量检测。

**方法三：第三方检测**

使用市面上专业的敏感词检测工具。

## 排版规范

### 基础排版

| 项目 | 推荐值 |
|-----|--------|
| 段落长度 | 三至五行 |
| 行间距 | 一点七五倍 |
| 字间距 | 默认 |
| 两端缩进 | 十六像素 |
| 字体大小 | 十五至十六像素 |

### 重点突出

**加粗使用**：

- 加粗核心关键词
- 每段加粗一至两个关键点
- 不要过度使用加粗

**变色使用**：

- 使用品牌色或强调色
- 重点信息使用不同颜色
- 颜色种类不超过两种

**emoji使用**：

- 适当使用emoji增加亲和力
- 不要过度使用
- 与内容风格匹配

**引用框使用**：

- 引用名言或重要观点时使用
- 区分观点和引用内容
- 增加文章层次感

### 配图原则

**配图类型**：

| 配图位置 | 配图类型 | 作用 |
|---------|---------|------|
| 开头配图 | 封面图、吸引注意图 | 吸引点击 |
| 过渡配图 | 缓解疲劳图 | 分割章节 |
| 结尾配图 | 引导互动图 | 促进互动 |

**配图规范**：

- 图片清晰度要高
- 图片风格要统一
- 图片大小要合适
- 图片版权要合规

**配图频率**：

- 每三百至五百字配一张图
- 长文章需要更多配图
- 配图要与内容相关

## 使用示例

### 示例：AI Agent趋势文章

**标题**：二零二六年AI Agent最新趋势：这五个变化正在改变一切

**开头**：

```
你有没有发现，二零二六年刚开年，"AI Agent"这个词突然无处不在了？

前几天，我一个在互联网大厂做产品的朋友跟我说，他们部门最近开会，"Agent"这个词出现的频率，比当年说"元宇宙"还要高。

不只是大厂。我认识的好几个创业者，都在把自己的项目往Agent方向靠。

说实话，我一开始有点懵。这玩意儿到底有什么魔力？值得所有人all in吗？

于是我花了整整一周时间，调研了国内外几十个Agent项目，翻了上百份行业报告，找了十几位业内人士聊了聊。

今天这篇文章，我想把看到的、听到的、思考的，全部分享给你。

【配图建议】：封面图使用科技感插画，标题醒目标注
```

**正文风格特点**：

- 口语化表达：如"这玩意儿"、"说实话"、"all in"
- 网感表达：如"破防了"、"绝绝子"等适度使用
- 场景代入：描述具体场景，引发读者共鸣
- 实用价值：提供五个趋势分析和具体案例

**金句运用**：

- "如果说ChatGPT是'嘴替'，那Agent就是'手替'"
- "未来不会用Agent的人，可能会像现在不会用电脑的人一样被动"
- "它不是替代你，而是放大你"

**结尾**：

```
好，以上就是我对二零二六年AI Agent趋势的观察和思考。

最后总结五个核心要点：

一、从聊天到干活：AI正在从回答问题的工具，变成能办事的代理

二、重新定义员工：每个行业都在引入Agent，提升效率、降低成本

三、身体比大脑更重要：光有强大的模型不够，关键是有好的执行和协作能力

四、Personal Agent成为新资产：属于自己的、训练好的Agent，将成为重要的数字资产

五、合规是核心竞争力：监管加速，合规能力将决定谁能走得更远

【互动引导】

你用过AI Agent吗？有什么特别的体验或者踩坑经历？

评论区聊聊，我每条都会看。

如果觉得这篇文章对你有帮助，点个在看吧，你的支持是我持续输出的动力。
```

## 常见问题

### 问题一：如何把握口语化的度？

**回答**：口语化不等于随意。要在亲切和专业之间找到平衡。关键场景可以用口语，日常叙述保持书面语的简洁。

### 问题二：网感表达如何使用？

**回答**：根据目标读者群体调整。如果读者是年轻人，可以适当使用网感表达；如果读者偏成熟，建议减少使用频率。

### 问题三：如何避免标题党？

**回答**：标题承诺的内容，正文一定要兑现。不要过度夸大或制造虚假预期。

### 问题四：文章长度多少合适？

**回答**：公众号文章建议字数在二千至三千字之间。短文章一千至一千五百字适合快讯、干货；长文章三千至五千字适合深度分析。

### 问题五：如何提高完读率？

**回答**：提高完读率的关键是：
- 开头要有吸引力
- 段落要短，阅读压力小
- 适当使用配图缓解疲劳
- 设置悬念，引导读者继续阅读
- 金句和亮点分散在文章各处
