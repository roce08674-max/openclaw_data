## 快速开始

### 基本概念

标题agent负责标题创作、优化和心理学原理研究。一个好的标题决定了文章的点击率，是爆款文章的关键要素。

### 工作流程

```
内容分析 → 模板匹配 → 心理学应用 → 候选生成 → 优化筛选
```

## 标题类型分类

### 1. 震惊体

**结构模板**
- 震惊！XXX竟然...
- 万万没想到，XXX...
- XXX内幕曝光，全场...
- 看完这个，XXX再也不敢...

**经典案例**
1. "震惊！这个中国人一句话，让美国损失千亿"
2. "看完这个视频，我默默删掉了手机里所有APP"
3. "央视曝光！这种鱼千万别吃，快告诉家人"
4. "内部消息：XXX即将大变，涉及几亿人"

**适用场景**
- 负面新闻、爆料
- 颠覆认知的信息
- 紧急提醒、警告

**使用建议**: 震惊体冲击力强，但容易被认为是标题党，建议配合实际有价值的内文使用

---

### 2. 数字体

**结构模板**
- XXX的N个真相/秘密/方法
- XXX天，XXX发生了N件大事
- 3分钟学会XXX，7天见效
- N个理由告诉你为什么...

**经典案例**
1. "月薪3000和月薪30000的区别，就在这一点"
2. "2024年最值得关注的10个趋势"
3. "5个技巧，让你的文章阅读量翻10倍"
4. "30岁前必须明白的5个道理"

**适用场景**
- 知识分享、干货
- 排行榜、清单
- 对比分析

**使用建议**: 数字体给人一种明确、可预期的感觉，适合干货类、技巧类内容

---

### 3. 疑问体

**结构模板**
- XXX真的XXX吗？
- 为什么XXX？答案在这里
- XXX到底是怎么做到的？
- XXX背后的真相是...

**经典案例**
1. "为什么他每天工作3小时，收入却是你的10倍？"
2. "月薪5000，凭什么能在上海活得很好？"
3. "年轻人为什么都不爱换手机了？"
4. "特斯拉降价背后，藏着什么秘密？"

**适用场景**
- 深度分析、解读
- 观点输出
- 问题解决型内容

**使用建议**: 疑问体引发读者思考，适合有明确答案或深度分析的内容

---

### 4. 情感体

**结构模板**
- 看完XXX，我哭了
- 致XXX：对不起，我爱你
- XXX的故事，让亿万人泪目
- 如果你也在经历XXX，请看完这篇

**经典案例**
1. "那个裸辞去大理的姑娘，后来怎么样了？"
2. "写给所有正在努力的人：别放弃"
3. "一位母亲的独白：对不起，是妈妈害了你"
4. "那个在最艰难时刻帮助我的人，我终于找到了"

**适用场景**
- 人物故事
- 情感共鸣
- 励志成长

**使用建议**: 情感体触发读者情绪，适合有故事性、有情感深度的内容

---

### 5. 悬念体

**结构模板**
- XXX的真相，99%的人都不知道
- XXX之后，发生了这件
- 看完这个故事，你会重新理解XXX
- XXX的秘密，今天终于被揭开

**经典案例**
1. "巴菲特私生活曝光，90%的人都没见过"
2. "他用了3年时间，只为做这一件事"
3. "这个视频里的细节，90%的人都没注意到"
4. "马云退出背后，隐藏着一个更大计划"

**适用场景**
- 深度报道
- 揭秘类内容
- 人物专访

**使用建议**: 悬念体利用信息差制造好奇心，但需要在正文中兑现承诺

---

### 6. 热点借势体

**结构模板**
- XXX上热搜，我只说3点
- 从XXX事件，看懂这3个趋势
- XXX刷屏后，我明白了这5个道理
- 聊聊XXX这件事

**经典案例**
1. "ChatGPT爆火，我看到了这3个机会"
2. "从《繁花》大火，看上海文化的复兴"
3. "雷军年度演讲，这5句话值得反复看"
4. "全红婵夺冠后，我看到了最真实的一面"

**适用场景**
- 热点事件解读
- 借势营销
- 趋势分析

**使用建议**: 热点借势体时效性强，需要快速反应，适合热点解读类内容

## 心理学原理

### 1. 好奇心缺口理论

**理论解释**: 人们看到信息缺口时，会产生强烈的好奇心，想要填补这个缺口。

**应用技巧**

1. **制造信息差**
   - "只有1%的人知道的秘密"
   - "XXX的真相，99%的人都不知道"

2. **设置悬念**
   - "他最后的选择，让所有人沉默"
   - "这个决定，改变了一切"

3. **提出问题**
   - "为什么聪明人都在做这件事？"
   - "XXX的真正原因是什么？"

**实战案例对比**

```
❌ 普通标题：如何提高工作效率
✅ 心理标题：为什么你工作8小时不如别人2小时？

❌ 普通标题：10个理财建议
✅ 心理标题：90%的人都在犯的理财错误
```

---

### 2. 损失厌恶理论

**理论解释**: 人们对损失的敏感度远高于对收益的敏感度，失去的痛苦大于获得的快乐。

**应用技巧**

1. **强调损失**
   - "失去这个，你会后悔一辈子"
   - "别让这个习惯毁掉你的孩子"

2. **制造紧迫感**
   - "最后24小时，错过再等一年"
   - "这篇文章即将被删除"

3. **威胁式标题**
   - "如果你还在用这个密码，小心被盗"
   - "这种食物千万别吃，会致癌"

**实战案例对比**

```
❌ 普通标题：保护眼睛的方法
✅ 心理标题：还在这样看手机？你的眼睛正在被毁掉

❌ 普通标题：理财基础知识
✅ 心理标题：90%的人都在亏损，原因是这个
```

---

### 3. 社会认同理论

**理论解释**: 人们倾向于参考他人的行为和意见来做决定。

**应用技巧**

1. **引用数据**
   - "10万人都在用的方法"
   - "3亿人都看过的视频"

2. **权威背书**
   - "清华教授推荐的学习方法"
   - "马斯克都在用的时间管理"

3. **从众效应**
   - "大家都在看的文章"
   - "2024年最火的XXX"

**实战案例对比**

```
❌ 普通标题：健身计划分享
✅ 心理标题：100万人跟随的健身计划有何不同？

❌ 普通标题：读书推荐
✅ 心理标题：比尔·盖茨每年必读的书单
```

---

### 4. 身份认同理论

**理论解释**: 人们关心与自己身份、价值观相关的信息。

**应用技巧**

1. **标签化**
   - "给XXX人群的专属建议"
   - "XX星座的你一定要看"

2. **价值观共鸣**
   - "只有XXX的人才懂"
   - "真正的XXX是什么样的"

3. **身份代入**
   - "如果你也是XXX，一定要看"
   - "XXX人必读的N条建议"

**实战案例对比**

```
❌ 普通标题：职场建议
✅ 心理标题：30岁之前必须明白的职场真相

❌ 普通标题：护肤技巧
✅ 心理标题：精致女孩都在用的护肤步骤
```

---

### 5. 情感触发理论

**理论解释**: 情感是驱动人们行动的最大动力。

**情感类型与应用**

1. **恐惧情感**
   - 焦虑、担忧、害怕
   - 触发词：危险、毁掉、失去、后悔

2. **好奇情感**
   - 想知道更多
   - 触发词：秘密、真相、内幕、揭秘

3. **贪婪情感**
   - 想获得更多
   - 触发词：免费、赚钱、省钱、暴富

4. **懒惰情感**
   - 想轻松获得
   - 触发词：简单、快速、轻松、一键

5. **归属情感**
   - 想被认同
   - 触发词：大家、所有人、群体

---

### 6. 认知负荷理论

**理论解释**: 人们的大脑倾向于处理简单的信息，复杂的信息会被过滤。

**应用技巧**

1. **降低认知负荷**
   - 标题要短（15-25字）
   - 结构要清晰
   - 关键词要突出

2. **使用简单词汇**
   - 避免专业术语
   - 使用常见词汇
   - 口语化表达

3. **突出核心价值**
   - 让读者一眼看懂
   - 立即知道收益
   - 降低决策成本

---

### 7. 注意力竞争理论

**理论解释**: 在信息过载的时代，标题必须足够吸引眼球才能获得关注。

**应用技巧**

1. **使用强动词**
   - "颠覆"、"引爆"、"震惊"、"炸裂"

2. **使用情绪词**
   - "泪目"、"炸裂"、"沸腾"、"疯了"

3. **使用数字**
   - 具体、量化、有冲击力的数字

4. **使用对比**
   - 前后对比
   - 强弱对比
   - 快慢对比

## 标题生成流程

### 流程步骤

#### 步骤1: 明确内容类型

确定文章属于哪种类型：
- 观点输出型
- 干货分享型
- 故事叙述型
- 热点解读型

#### 步骤2: 选择标题模板

根据内容类型选择合适的标题模板（参考标题类型分类）

#### 步骤3: 应用心理学原理

运用至少2-3个心理学原理优化标题

#### 步骤4: 生成候选标题

至少生成5-8个候选标题

#### 步骤5: 主标题选择

综合评估，选择最优标题作为主标题

### 标题优化清单

- [ ] 控制在15-25字之间
- [ ] 包含数字或具体信息
- [ ] 使用强动词或情绪词
- [ ] 突出读者收益
- [ ] 引起好奇心
- [ ] 降低认知负荷
- [ ] 无违规词
- [ ] 符合平台规则

## 避坑指南

### 违规词黑名单

**绝对化用语**
```
绝对、最好、第一、顶级、唯一、史上最强、顶尖
```

**夸张用语**
```
震惊、全惊呆、吓尿、哭晕、必看、必读、必须知道
```

**虚假承诺**
```
100%有效、无效退款、一招见效、保证赚钱
```

**违规承诺（用于虚假宣传时）**
```
独家、揭秘、内幕、真相
```

### 平台禁忌

**微信公众平台**
- 禁止标题党
- 违规会限流
- 虚假宣传会封号

**抖音**
- 禁止夸大、虚假信息
- 禁止诱导互动
- 禁止标题党和封面党

**小红书**
- 禁止诱导互动
- 禁止虚假种草
- 禁止夸大宣传

**微博**
- 禁止攻击、谣言
- 禁止引战
- 禁止违规营销

### 低质量标题特征

1. **过于平淡**
   - 示例："今天吃什么"
   - 问题：没有吸引力

2. **信息模糊**
   - 示例："这件事你怎么看"
   - 问题：读者不知道能得到什么

3. **过于专业**
   - 示例："论XXX的YYY属性"
   - 问题：认知负荷太高

4. **太长或太短**
   - 超过25字或在5字以内
   - 问题：要么信息过载，要么信息不足

5. **关键词堆砌**
   - 示例："苹果手机iPhone手机最新款手机"
   - 问题：看起来像垃圾信息

## 使用示例

### 示例：AI Agent热点文章标题生成

**内容背景**: 2026年AI Agent成为科技圈热点话题

**候选标题生成**:

震惊体:
1. "2026年第一个超级风口诞生！AI Agent正在颠覆整个行业"
2. "看完这个，我终于明白为什么所有人都在聊AI Agent"

数字体:
3. "2026年AI Agent最新趋势：这5个变化正在改变一切"
4. "3分钟读懂AI Agent：为什么它能火遍全网？"

悬念体:
5. "AI Agent爆火背后，藏着下一个万亿级市场"
6. "这个让科技巨头集体押注的技术，到底有什么魔力？"

疑问体:
7. "为什么2026年所有科技公司都在布局AI Agent？"
8. "AI Agent会成为下一个iPhone时刻吗？"

**评估与选择**:

| 标题 | 心理学原理 | 吸引力 | 合规性 | 综合分 |
|------|-----------|--------|--------|--------|
| 震惊体1 | 好奇心、损失厌恶 | 9 | 8 | 8.5 |
| 震惊体2 | 好奇心、社会认同 | 8 | 9 | 8.5 |
| 数字体1 | 数字、好奇心 | 9 | 10 | 9.5 |
| 数字体2 | 数字、懒惰 | 8 | 9 | 8.5 |
| 悬念体1 | 好奇心、贪婪 | 9 | 9 | 9.0 |
| 悬念体2 | 好奇心、社会认同 | 8 | 9 | 8.5 |
| 疑问体1 | 好奇心、社会认同 | 8 | 10 | 9.0 |
| 疑问体2 | 好奇心、对比 | 9 | 9 | 9.0 |

**推荐主标题**: "2026年AI Agent最新趋势：这5个变化正在改变一切"

**选择理由**:
- 数字体有明确的预期管理
- 符合干货分享型内容定位
- 无任何违规词
- 适合目标读者群体（科技从业者、创业者）
