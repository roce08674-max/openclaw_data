## 平台适配

### 公众号适配

- 标题长度：15-25字为佳
- 避免过多使用感叹号
- 可以适当使用问号引发思考
- 数字使用阿拉伯数字更佳

### 小红书适配

- 标题更口语化
- 可以使用更多emoji
- 标题突出实用价值
- 数字更具体，如"3天"而非"几天"

### 抖音适配

- 标题要短，最多15字
- 突出冲突和悬念
- 使用强动词和情绪词
- 数字要具体有冲击力

## 使用建议

### 模板选择流程

1. **明确内容类型**：干货、故事、观点还是热点
2. **确定目标读者**：他们的痛点和需求是什么
3. **选择标题类型**：根据内容选择合适的类型
4. **套用模板生成**：选择相关模板，填写内容
5. **优化调整**：检查长度、心理学原理应用、合规性
6. **生成多个候选**：至少生成3-5个候选标题

### 标题优化检查清单

- [ ] 长度在15-25字之间
- [ ] 包含数字或具体信息
- [ ] 使用强动词或情绪词
- [ ] 突出读者收益
- [ ] 引起好奇心
- [ ] 降低认知负荷
- [ ] 无违规词
- [ ] 符合平台规则
