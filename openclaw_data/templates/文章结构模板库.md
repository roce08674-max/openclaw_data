# 文章结构模板库

本文档收集整理了各类文章结构模板，可根据内容类型选择使用。

---

## 一、观点输出型结构

### 模板概述

**适用场景**：观点类、评论类、思考类文章
**篇幅**：1500-2500字
**结构**：总分总模式

### 具体结构

```
【开头】（150-200字）
场景引入/现象描述
↓
提出核心观点
↓
预告文章价值

【正文】（1200-1800字）
分论点1 + 案例支撑（400-600字）
过渡段（100-150字）
分论点2 + 案例支撑（400-600字）
过渡段（100-150字）
分论点3 + 案例支撑（400-600字）

【结尾】（150-200字）
总结核心观点
↓
升华主题
↓
引导互动
```

### 示例

```
开头：同事小王工作3年还是月薪5000，而他只做了一件事...
观点：选择比努力更重要，但努力是为了有更多选择

分论点1：选择决定方向——方向错了，再努力也是徒劳
案例：小张和小李，同样的努力，不同的选择...

分论点2：选择需要智慧——不是碰运气，而是有方法
案例：如何做出正确选择？3个实用方法...

分论点3：选择需要勇气——有时候，放弃也是一种选择
案例：裸辞创业的人，后来都怎么样了？

结尾：愿你既有努力的能力，也有选择的智慧
```

**篇幅**：2000-3500字
**结构**：问题-方案模式

```
描述痛点场景
↓
引出解决方案
↓
承诺文章价值

【正文】（1800-2800字）
方法1：核心要点 + 详细步骤 + 注意事项 + 案例演示（500-700字）
方法2：核心要点 + 详细步骤 + 注意事项 + 案例演示（500-700字）
方法3：核心要点 + 详细步骤 + 注意事项 + 案例演示（500-700字）
常见误区/坑点（200-300字）

【结尾】（100-150字）
总结核心要点
↓
鼓励立即行动
↓
资源推荐/互动引导
```

## 使用建议

### 结构选择考量因素

1. **内容类型**：纯观点、干货教程、故事、热点、资源整理、对比分析
2. **目标读者**：读者群体的阅读习惯和偏好
3. **传播目的**：表达观点、传递知识、引发讨论
4. **篇幅要求**：根据平台限制选择合适的结构

### 结构优化技巧

1. **开头要抓人**：用场景、问题、数据或悬念开头
2. **过渡要自然**：用过渡句连接各部分
3. **重点要突出**：重要内容要详细展开
4. **结尾要有力**：总结观点，升华主题，引导行动
