## 二、图片生成 Agent

**路径**: `skills/image-agent/`

| 属性 | 值 |
|------|-----|
| **名称** | 图文 Agent (Image Agent) |
| **核心职能** | 为公众号文章生成高质量配图 |
| **使用 API** | 华为云 Gitee AI (Z-Image 模型) |
| **默认尺寸** | 1024x1024 |
| **每日限制** | 100 张 / 天 (UTC 0:00 重置) |

### 主要功能

- **智能配图生成**: 根据文章内容自动生成匹配图片，支持多种风格和尺寸定制
- **使用量监控**: 严格遵循每日 100 张的限制，自动跟踪使用量并提醒
- **高质量输出**: 支持 1024x1024 高清分辨率，满足公众号配图需求
- **批量生成**: 支持多个提示词批量生成
- **提示词优化**: 根据风格（科技、商业、生活方式）自动优化提示词

### 技术架构

| 组件 | 技术 |
|------|------|
| **API 提供商** | 华为云 Gitee AI |
| **模型** | Z-Image |
| **接口方式** | OpenAI 兼容 API |
| **语言** | Python |
| **依赖** | openai, requests, base64, python-dotenv |

### API 配置（安全方式）

```python
# 方式1：使用环境变量（推荐）
import os
from dotenv import load_dotenv

load_dotenv()
api_key = os.environ.get("GITEE_AI_API_KEY")

# 方式2：传入参数
agent = ImageAgent(api_key="your_api_key_here")

# 方式3：.env 文件
# 创建 .env 文件，内容如下：
# GITEE_AI_API_KEY=your_api_key_here
```

### 使用规则

**每日限制**:

| 规则 | 值 | 说明 |
|------|-----|------|
| **每日上限** | 100 张 | UTC 0:00 重置 |
| **单次请求** | 1 张 | 默认为 1 张 |
| **超出处理** | 拒绝请求 | 返回友好错误 |

**速率限制**（遵循 NVIDIA API 规则）:

| 限制项 | 值 | 应对策略 |
|--------|-----|---------|
| **RPS** | 1-2 | 控制请求间隔 ≥0.5 秒 |
| **RPM** | 60 | 足够日常使用 |
| **并发数** | ≤5 | 使用信号量控制 |
| **HTTP 429** | 速率限制 | 指数退避（1s→2s→4s→8s） |

### 使用示例

```python

    prompt="AI 科技感办公室，高清专业",
    save_path="output/tech-office.jpg"
```

## 三、社交媒体分析器

**路径**: `skills/social-media-analyzer/`

| 属性 | 值 |
|------|-----|
| **名称** | Social Media Analyzer |
| **核心职能** | 分析社交媒体 campaign 表现和 ROI |
| **支持平台** | Instagram, Facebook, Twitter/X, LinkedIn, TikTok |

### 主要功能

- **参与度指标计算**: 包含点赞、评论、分享、收藏等
- **ROI 计算**: 投资回报率分析
- **平台对比**: 与行业基准进行比较
- **内容评估**: 识别表现最佳和最差的内容
- **完整分析报告**: 生成详细的性能分析

### 核心指标公式

**参与率 (Engagement Rate)**:

```
参与率 = (点赞 + 评论 + 分享 + 收藏) / 触达 × 100
```

**每次互动成本 (CPE)**:

```
CPE = 总花费 / 总互动数
```

**每次点击成本 (CPC)**:

```
CPC = 总花费 / 总点击数
```

**投资回报率 (ROI)**:

```
ROI = (价值 - 花费) / 花费 × 100
```

### 参与率基准

| 平台 | 平均 | 良好 | 优秀 |
|------|------|------|------|
| Instagram | 1.22% | 3-6% | >6% |
| Facebook | 0.07% | 0.5-1% | >1% |
| Twitter/X | 0.05% | 0.1-0.5% | >0.5% |
| LinkedIn | 2.0% | 3-5% | >5% |
| TikTok | 5.96% | 8-15% | >15% |

### 使用方式

```bash
# 分析 campaign 表现
python scripts/analyze_performance.py assets/sample_input.json

# 计算指标
python scripts/calculate_metrics.py assets/sample_input.json
```
